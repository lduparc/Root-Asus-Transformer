Root/Restore Instructions - Asus Transformer

You can find two sections to root your tablet. To find your android version, go to settings on your tablet, about section and find your Android Version.

  • Requierement

For all version of your Asus Transformer, you must have Android SDK tools installed Download ANDROID SDK ( http://developer.android.com/sdk/index.html )

  • Install tools from ANDROID SDK

Go to your android-sdk directory and tools

On Mac OS X ( and other Unix platform ):

Install android tools ./android ( select tools and install )

On Windows: Simply double click on android file, select tools and install.

Unroot/Restore Asus Transformer

These instructions is the easiest way to unroot and restore your Asus Transformer. At this time, it just work under Windows.

  • Requierement

Download project archive into zip or tar.gz format. This archive content nvFlashWW directory. Your tablet must not be plug to your computer at beginning.

  • Unroot/Restore your tablet

Start your Asus Transformer by pushing power button and Volume ( + ) during 8 seconds; Your screen staying black, it's normal.

Plug it to your computer. Windows automatically detect your tablet. If Windows notify that it can't install driver, go to "Driver Manager", select "Update driver on APX device". Then select update from computer and select nvFlashWW/usbpcdriver directory, check "find in subdirectories".

Go to nvFlashWW directory and launch download bat file. Waiting reboot.

You have restored your tablet.

Root Version TF101/TF201 (Prime) v4.0.x

These instructions is the easiest way to root Asus Transformer for Android 4.0.x. If you have an android version in 3.x got to Version TF101 3.x section or upgrade your tablet to android 4.0.x. It just work under Windows operating system.

  • Requierement

Download project archive into zip or tar.gz format. This archive content viperMOD PrimeTime v4.5 directory.

Your tablet must not be plug to your computer at beginning.

  • Root your tablet

Start your Asus Transformer by pushing power button and Volume ( + ) during 8 seconds; Your screen staying black, it's normal.

Plug it to your computer. Windows automatically detect your tablet. If Windows notify that it can't install driver, go to "Driver Manager", select "Update driver on APX device". Then select update from computer and select viperMOD PrimeTime v4.5/usbpcdriver directory, check "find in subdirectories".

After drivers installed, go to viperMOD PrimeTime v4.5 and launch viperMOD PrimeTime v4.5 executable.

Select correct option ( must be option 1 ).

Follow instructions ( just press any key to continue at each step ).

When your Asus Transformer is rebooted, your tablet must be rooted.

  • Finally

Download superUser and Voodoo OTA RootKeeper from market to test root and activate it. Voodoo OTA RootKeeper allow to backup root mode and activate or desactivate root mode to do some update. Maybe you could wait some minutes during Android activate it.

Root Version TF101 3.x

These instructions is the easiest way to root Asus Transformer for Android 3.x. This not work on Android 4.x

Download project archive into zip or tar.gz format. This archive content nachoroot.bin and su script.

  • Root your tablet Go to android-sdk directory and platform-tools
adb push nachoroot.bin /data/local/

You must specify the path where you have downloaded the project archive. Example : adb push /Users/your_name/Desktop/nachoroot.bin /data/local/

adb shell chmod 777 /data/local/nachoroot.bin
adb shell /data/local/nachoroot.bin --stage1
adb reboot

Wait during your tablet is rebooting

adb shell /data/local/nachoroot.bin --stage2
adb reboot

Wait during your tablet is rebooting

adb shell id

If id is different to 0, start over root instructions

adb remount
adb  push  su  /system/xbin/su

You must specify the path where you have downloaded the project archive. Example : adb push /Users/your_name/Desktop/su /system/xbin/su

adb shell chown 0.0 /system/xbin/su
adb shell chmod 06755 /system/xbin/su
adb shell /data/local/nachoroot.bin --undo
adb reboot

Wait during your tablet is rebooting

  • Finally

Download superUser and Voodoo OTA RootKeeper from market to test root and activate it. Voodoo OTA RootKeeper allow to backup root mode and activate or desactivate root mode to do some update. Maybe you could wait some minutes during Android activate it.
